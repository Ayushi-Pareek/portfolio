DeepSource · 2019–2021

Making code review findings impossible to ignore

Role Product Designer
Scope Product redesign, design system
Team Design, engineering, founders

Overview

A static analysis tool that needed to earn developer trust

DeepSource automates code quality checks, surfacing issues across pull requests before they reach production. I joined as the design lead and worked directly with the founders to redesign the product, build the brand, and create Zeal, DeepSource's component-based design system, from scratch.
